ВАРІАНТ 1: REPLIT (НАЙПРОСТІШИЙ) ⭐ РЕКОМЕНДУЄТЬСЯ

1. Перейдіть на replit.com
2. Натисніть "Create Repl"
3. Виберіть Python
4. Скопіюйте код telegram_bot.py у файл main.py
5. Скопіюйте код requirements.txt
6. Натисніть "Add file" → ".env"
7. Додайте змінні:
   TELEGRAM_TOKEN=ваш_токен_від_BotFather
   ANTHROPIC_API_KEY=ваш_api_key

8. Натисніть "Run" (зеленою кнопкою)
9. Бот буде запущено і працюватиме 24/7!

⚠️ Важливо: Replit буває вимикає вільні репли після неактивності
   → Купіть Replit Plus ($7/міс) для безперервної роботи ІЛИ
   → Используйте другу опцію ⬇️

ВАРІАНТ 2: RAILWAY.APP

1. Перейдіть на railway.app
2. Натисніть "New Project"
3. Виберіть "Deploy from GitHub"
   (або завантажте файли вручну)
4. Додайте environment variables:
   - TELEGRAM_TOKEN
   - ANTHROPIC_API_KEY
5. Додайте Procfile:
   ┌──────────────────────────────────────┐
   │ worker: python telegram_bot.py       │
   └──────────────────────────────────────┘
6. Натисніть Deploy
7. ✅ Бот запущено!

Безплатно: 500 годин/місяць
Платна версія: $5/місяць за додаткові ресурси

ВАРІАНТ 4: VPS (CONTABO, DIGITALOCEAN, HETZNER)

1. Оберіть сервіс:
   • Contabo: 2.99€/місяць
   • DigitalOcean: $4/місяць
   • Hetzner: дешевше

2. Підключіться через SSH:
   ssh root@ваш_ip

3. Встановіть Python:
   apt update
   apt install python3 python3-pip

4. Завантажте файли:
   git clone <ваш_репозиторій>
   cd <папка>

5. Встановіть залежності:
   pip install -r requirements.txt

6. Створіть .env файл:
   cat > .env << EOF
   TELEGRAM_TOKEN=ваш_токен
   ANTHROPIC_API_KEY=ваш_key
   EOF

7. Запустіть бота:
   python3 telegram_bot.py

8. Щоб бот працював 24/7, використайте screen або systemd:
   
   ┌──────────────────────────────────────┐
   │ Варіант А: Screen (просто)           │
   │ screen -S bot                        │
   │ python3 telegram_bot.py              │
   │ Ctrl+A потім D щоб вийти             │
   └──────────────────────────────────────┘

   ┌──────────────────────────────────────┐
   │ Варіант Б: Systemd (краще)           │
   │ Створіть файл:                       │
   │ /etc/systemd/system/bot.service      │
   └──────────────────────────────────────┘

   Вміст bot.service:
   ════════════════════════════════════════
   [Unit]
   Description=Telegram Code Agent Bot
   After=network.target

   [Service]
   Type=simple
   User=root
   WorkingDirectory=/root/telegram_bot
   Environment="PATH=/usr/local/bin:/usr/bin"
   ExecStart=/usr/bin/python3 telegram_bot.py
   Restart=always
   RestartSec=10

   [Install]
   WantedBy=multi-user.target
   ════════════════════════════════════════

   Активуйте:
   systemctl start bot
   systemctl enable bot
   systemctl status bot

🔧 НАЛАГОДЖЕННЯ:

Якщо виникли проблеми:

1. Перевірте змінні середовища:
   echo $TELEGRAM_TOKEN
   echo $ANTHROPIC_API_KEY

2. Перевірте логи (залежно від сервісу):
   Replit: видно в консолі
   Railway: вкладка "Logs"
   VPS: tail -f /var/log/syslog

3. Частіші помилки:
   ❌ "Token invalid" → перевірте TELEGRAM_TOKEN
   ❌ "API key not found" → перевірте ANTHROPIC_API_KEY
   ❌ "No module named telegram" → pip install -r requirements.txt
   ❌ "Connection timeout" → перевірте інтернет-з'єднання
