ШАГ 1: СТВОРІТЬ БОТА В TELEGRAM (2 ХВИЛИНИ)
───────────────────────────────────────────

Откройте Telegram приложение на телефоне:

1. Знайдіть контакт "BotFather"
2. Напишіть: /newbot
3. Введіть ім'я (наприклад: CodeAgentBot)
4. Введіть username (наприклад: code_agent_bot_123)
5. Скопіюйте TOKEN (це довгий рядок з цифр і букв)

✅ TOKEN ГОТОВИЙ!

═══════════════════════════════════════════════════════════════

ШАГ 2: ОТРИМАЙТЕ API KEY (2 ХВИЛИНИ)
──────────────────────────

1. Перейдіть на console.anthropic.com
2. Залогіньтесь (створіть аккаунт якщо потрібно)
3. Натисніть "API Keys" 
4. Натисніть "Create new API key"
5. Скопіюйте ключ (починається на sk-ant-)

✅ API KEY ГОТОВИЙ!

═══════════════════════════════════════════════════════════════

ШАГ 3: РОЗГОРНІТЬ НА REPLIT (5 ХВИЛИН) - НАЙПРОСТІШЕ!
────────────────────────────────────────

1. Перейдіть на replit.com
2. Натисніть "+ Create" → "Create Repl"
3. Виберіть мову: Python
4. Натисніть "Create Repl"

5. У лівій панелі натисніть "+" біля "Files"
6. Додайте файл: main.py
   - Скопіюйте весь код з telegram_bot.py

7. Додайте файл: requirements.txt
   - Скопіюйте:
     python-telegram-bot==21.1
     anthropic==0.31.0
     aiohttp==3.9.1
     python-dotenv==1.0.0

8. Натисніть іконка "Secrets" (замок) в нижній частині
9. Додайте змінні:
   - Key: TELEGRAM_TOKEN
     Value: [ваш TOKEN від BotFather]
   
   - Key: ANTHROPIC_API_KEY
     Value: [ваш API KEY з Anthropic]

10. Натисніть зелену кнопку "Run"
11. Чекайте "Bot is now running..."

✅ БОТ ЗАПУЩЕНО! 🚀

═══════════════════════════════════════════════════════════════

ШАГ 4: НАПИШІТЬ БОТУ В TELEGRAM
──────────────────────────────

1. Откройте Telegram
2. Знайдіть свого бота (за username який ви створили)
3. Напишіть: /start
4. Слідуйте інструкціям бота!

СПРОБУЙТЕ:
• /apis - побачити список API
• "Напиши функцію для отримання погоди"
• "Як інтегрувати Spotify API?"
• "Покажи код для крипто-трекера"

✅ ВСЕ ПРАЦЮЄ! 🎉

Допомога: /help

🆘 ПРОБЛЕМИ?

❌ Проблема: "No module named telegram"
✅ Рішення: Натисніть "Run" знову, дайте 1 хвилину на установку

❌ Проблема: "Invalid token" 
✅ Рішення: Перевірте що скопіювали ВЕСЬ TOKEN правильно

❌ Проблема: Бот не відповідає
✅ Рішення: 
  1. Перевірте що натиснули /start
  2. Дайте боту 30 секунд на відповідь (перший запит повільний)
  3. Напишіть /help
