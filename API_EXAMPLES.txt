🌤️  OPENWEATHER API - ОТРИМАННЯ ПОГОДИ

Напишіть боту:
"Напиши функцію для отримання поточної погоди з OpenWeather API"

Бот напише щось подібне:

```python
import requests

def get_weather(city, api_key):
    """
    Отримує поточну погоду для міста
    api_key отримати на: https://openweathermap.org/api
    """
    url = f"https://api.openweathermap.org/data/2.5/weather"
    params = {
        'q': city,
        'appid': api_key,
        'units': 'metric'  # Цельсій
    }
    
    response = requests.get(url, params=params)
    data = response.json()
    
    if response.status_code == 200:
        return {
            'temperature': data['main']['temp'],
            'description': data['weather'][0]['description'],
            'humidity': data['main']['humidity'],
            'wind_speed': data['wind']['speed']
        }
    else:
        return {'error': data['message']}

# Приклад використання:
api_key = 'ваш_api_key'
weather = get_weather('Київ', api_key)
print(weather)
```

💰 COINMARKETCAP API - КРИПТО ЦІНИ

Напишіть боту:
"Показ топ 10 крипто валют по ціні"

Бот напише:

```python
import requests

def get_top_crypto(limit=10, api_key=''):
    """Отримує топ крипто-валют"""
    url = 'https://pro-api.coinmarketcap.com/v1/cryptocurrency/listings/latest'
    
    headers = {
        'X-CMC_PRO_API_KEY': api_key,
    }
    
    params = {
        'limit': limit,
        'convert': 'USD'
    }
    
    response = requests.get(url, headers=headers, params=params)
    data = response.json()
    
    for crypto in data['data']:
        print(f"{crypto['name']}: ${crypto['quote']['USD']['price']:.2f}")

get_top_crypto()
```

API_KEY: https://pro.coinmarketcap.com/

🐙 GITHUB API - ПОШУК РЕПОЗИТОРІЇВ

Напишіть боту:
"Як пошукати популярні репозиторії на GitHub?"

Бот напише:

```python
import requests

def search_github_repos(language='python', stars='>=1000', limit=10):
    """Пошук репозиторіїв на GitHub"""
    url = 'https://api.github.com/search/repositories'
    
    params = {
        'q': f'language:{language} stars:{stars}',
        'sort': 'stars',
        'order': 'desc',
        'per_page': limit
    }
    
    response = requests.get(url, params=params)
    repos = response.json()['items']
    
    for repo in repos:
        print(f"⭐ {repo['name']}")
        print(f"   {repo['description']}")
        print(f"   Зірок: {repo['stargazers_count']}")
        print()

search_github_repos()
```

Без API_KEY для базового використання!

📸 PEXELS API - БЕЗПЛАТНІ ФОТО

Напишіть боту:
"Напиши функцію для завантаження фото з Pexels"

Бот напише:

```python
import requests

def get_photos(query, api_key, per_page=10):
    """Отримує фото з Pexels"""
    url = 'https://api.pexels.com/v1/search'
    
    headers = {'Authorization': api_key}
    params = {
        'query': query,
        'per_page': per_page
    }
    
    response = requests.get(url, headers=headers, params=params)
    data = response.json()
    
    for photo in data['photos']:
        print(f"Фото: {photo['photographer']}")
        print(f"URL: {photo['src']['medium']}")

get_photos('sunset', 'api_key')
```

API_KEY: https://www.pexels.com/api/

🌍 REST COUNTRIES API - ДАННІ ПРО КРАЇНИ

Напишіть боту:
"Напиши функцію для отримання даних про країну"

Бот напише:

```python
import requests

def get_country_info(country_name):
    """Отримує інформацію про країну"""
    url = f'https://restcountries.com/v3.1/name/{country_name}'
    
    response = requests.get(url)
    data = response.json()[0]
    
    return {
        'name': data['name']['common'],
        'population': data['population'],
        'capital': data['capital'][0] if data.get('capital') else 'N/A',
        'region': data['region'],
        'languages': data.get('languages', {}),
        'flag': data['flag']
    }

info = get_country_info('Ukraine')
for key, value in info.items():
    print(f"{key}: {value}")
```

БЕЗПЛАТНО, без API KEY!

def get_joke():

        print(f"Delivery: {data['delivery']}")
